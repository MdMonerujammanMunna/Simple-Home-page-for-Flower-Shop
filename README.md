# Simple-Home-page-for-Flower-Shop 

Home Page Design for a Flower Shop

A navigation bar has been used on the homepage.
On the left side, there is a logo, and on the right side, there are the following menu items: Home, Heading, Gallery, About, Blog, Shop, and Contact Info.
The navigation bar and text are made responsive, using a breakpoint of 880px.

A header titled “New Flower Shop” has been used on the homepage.
The background color is black, and the text color is apple green.
The font size of the header is 2rem (64px).

Below the header, some demo text has been added with a font size of 1.2rem (19px).
At the bottom, there is a button labeled “Read More” with the following styles:

Background color: apple green

Font size: 1rem (16px)

Font color: white

The pages has been created using "Tailwind css and HTML"------>Design & Code by Munna
